# Noubti Smart Queue

Noubti is a simple queue management app for small businesses. Customers can scan a QR code, choose a service, and take a digital ticket. Business owners can manage the queue from a dashboard.

## Features
- Public business page
- Ticket creation
- Ticket status tracking
- Waiting time calculation
- Service management
- QR code generation
- Simple daily statistics

## Technologies
- HTML
- CSS
- JavaScript
- Python
- Flask
- SQLite
- Jinja templates
